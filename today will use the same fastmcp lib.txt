##Steps for a simple remote server

1. Install uv
2. Create new folder
3. Open the folder in vs code
4. Uv init .
5. Uv add fastmcp
6. Create a simple server
7. Run the server uv run main.py
8. Test the server using mcp inspector
9. Create a GitHub repo
10. Git init
11. git add .
12. git commit -m "Initial commit: Simple MCP server"
13. git remote add origin [https://github.com/yourusername/simple-mcp-server.git](https://github.com/yourusername/simple-mcp-server.git)
14. git push -u origin main
15. Create an account on FastMCP cloud
16. Deploy on FastMCP cloud
